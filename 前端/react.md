# React简介

## 1.React是什么

用于构建用户界面的JavaScript库。

# Hello World

最简易的React示例如下：

```react
ReactDOM.render(
  <h1>Hello, world!</h1>,
  document.getElementById('root')
);
```

它将在页面上展示一个“Hello World！”的标题。

# jsx简介

考虑如下变量声明

```react
const element = <h1>Hello, world!</h1>;
```

既不是字符串也不是HTML。它被称为JSX，是一个个JavaScript的语法扩展。
